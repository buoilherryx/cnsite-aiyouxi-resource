# cnsite-aiyouxi-resource

## 项目简介

本仓库用于归档和发布多个独立 HTML 页面，旨在整理和共享一些互联网上收集或自制的网页资源。这些页面以静态 HTML 文件的形式存储，不依赖后端服务，可直接在浏览器中打开使用。

仓库内容不针对任何特定域名或网站，仅作为资源存档与分发用途。

## 目录说明

仓库根目录下包含以下内容：

- `index.html` — 项目主页，提供页面导航或说明。
- `pages/` — 存放所有独立 HTML 页面的文件夹。
- `assets/` — 页面使用的静态资源，如 CSS、JavaScript、图片等（如有）。
- `README.md` — 本文件。

## 页面归档说明

- 每个页面均为独立的 HTML 文件，可直接双击打开或在 Web 服务器上部署。
- 页面内容可能包含来自互联网的公开资源或自行创作的内容，仅供学习、参考或存档使用。
- 页面之间互不依赖，可单独使用或删除。

## 维护说明

- 本仓库不定期更新，新增或修改页面时会同步更新 `index.html` 导航链接。
- 欢迎通过 Issue 或 Pull Request 提交建议、修复或新增页面，但请确保内容合法合规。
- 仓库维护者保留对内容的筛选与修改权。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 打开 `index.html` 查看页面列表。
3. 点击链接或直接打开 `pages/` 下的 HTML 文件即可浏览。
